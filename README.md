# HMS-LINCS Small Molecule Suite Maintenance 

This repository contains scripts to process and maintain the data underlying the
Small Molecule Suite project, that was developed as part of the [NIH LINCS
Consortium](http://www.lincsproject.org/) in a collaboration between Harvard
Medical School's [Sorger Lab](http://sorger.med.harvard.edu/) and [Laboratory of
Systems Pharmacology
(LSP)](http://hits.harvard.edu/the-program/laboratory-of-systems-pharmacology/about/)
and the University of Cincinnati's [Laboratory for Statistical Genomics and
Systems Biology](http://eh3.uc.edu/). The former (HMS) is one of the [LINCS Data
and Signature Generation Centers
(DSGCs)](http://www.lincsproject.org/LINCS/centers/data-and-signature-generating-centers)
and the latter (Cincinnati) is part of the [BD2K-LINCS Data Coordination and
Integration Center (DCIC)](http://lincs-dcic.org/).

Publication:

Nienke Moret, Nicholas A. Clark, Marc Hafner, Yuan Wang, Eugen Lounkine, Mario Medvedovic, Jinhua Wang, Nathanael Gray, Jeremy Jenkins, Peter K. Sorger. <a href = "https://www.cell.com/cell-chemical-biology/fulltext/S2451-9456(19)30073-X" target="_blank">Cheminformatics Tools for Analyzing and Designing Optimized Small-Molecule Collections and Libraries.</a> *Cell Chem Biol* (2019). doi:10.1016/j.chembiol.2019.02.018
